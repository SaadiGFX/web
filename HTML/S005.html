That one is really simple to create. I used only line () & rect () functions from 2D Drawing.

line () function is used to draw the name (SAADI). And rect () function is used to draw the random colorful background.

On start the starting x co-ordinate of rect () function is incremented by value 2. But, when it touches the end-point of

the canvas, it start decrementing by value 5. Also, stroke () & random () functions are used.
